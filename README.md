Say hello to **Vertex**, the ultimate Quality-of-Life (QoL) evolution of the MacBook notch! Vertex is a high-performance fork of Boring Notch, specifically redesigned for power users, coders, and students who want their notch to be a functional command center rather than just a status bar.

Developed by an 11-year-old coder in Turkey 🇹🇷, Vertex takes the foundation of Boring Notch and aims to complete the "unfinished" features to create the most powerful Dynamic Notch experience on macOS.

## 📋 The Vertex Roadmap
We are focusing on checking the boxes that the original project left open, plus adding our own Vertex-exclusive features.

- [x] Playback live activity 🎧
- [x] Calendar & Reminders integration 📆
- [x] Charging indicator and percentage 🔋
- [x] Shelf functionality with AirDrop 📚
- [x] System HUD replacements (Volume/Brightness) 🎚️
- [ ] **Bluetooth Live Activity** (Real-time device tracking) 🔵 `NEW`
- [ ] **Weather Integration** (Live temp in the apex) ⛅️ `NEW`
